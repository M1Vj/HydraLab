# Attribution

Hydra is an original project that may study open-source tools as temporary references. Upstream repositories must not be committed as forks or runtime dependencies unless a future approved branch explicitly changes that rule.

## References

- T3 Code (`pingdotgg/t3code`) is MIT-licensed and may inspire workspace layout and AI-tool interaction patterns.
- Hermes Agent is a reference for skill-system, tool and memory concepts. Phase 1 does not depend on Hermes at runtime.

## License Preservation Rule

If Hydra copies or substantially adapts MIT-licensed code, preserve copyright notices and license text required by that project. Add file-level or document-level attribution where copied material appears.
